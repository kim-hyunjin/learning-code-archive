CommonDialog

how to use

파라미터

CommonDialog(title, options)

    title: string - 대화창 상단 헤더 제목. 기본값: ""
    options: object - 대화창 설정 옵션(optional)

options

contents: content[] - 대화창에 나타낼 항목(content) 목록
content: {
    label: string; // 항목명. 기본값: ""
    text: string; // 항목값. 기본값: ""
    editable: boolean; // 항목 편집 가능여부. 기본값: true
    computed: Function; // 계산되는 항목값(optional). 함수의 결과가 항목값으로 사용됩니다.
}

예제

import { CommonDialog } from "./common/dialog.js";

    const dialog1 = CommonDialog("예제1", {
        contents: [
            {
                label: "이름",
                text: "김현진",
                editable: false,
            },
            {
                label: "나이",
                text: 30,
                editable: false,
            },
            {
                label: "이메일",
                text: "hyunjin1612@gmail.com",
            },
            {
                label: "자기소개",
            },
        ],
    });

    const dialog2 = CommonDialog("예제2", {
        contents: [
            {
                label: "날짜",
                computed: function () {
                    const now = new Date();
                    return `${now.getFullYear()}-${now.getMonth() + 1}-${now.getDate()}`;
                },
                editable: false,
            },
        ],
    });

API

항목 전달인자 반환값 설명
open() - - 대화창을 표시합니다
close() - - 대화창을 닫습니다.
changeTitle() String: title - 대화창 상단 헤더 제목을 변경합니다.
editable() Boolean: isEditable Boolean: isEditable Dialog의 조회/편집 모드를 전환(토글)합니다.
save() - - 항목값들을 저장하고 조회 모드로 돌아갑니다.
cancel() - - 항목값들의 수정 사항을 취소하며, 조회 모드로 돌아갑니다.
getDataSource() - Array: contents 대화창에 표시되는 제목, 항목들의 정보를 리턴합니다.
setDataSource() Array: contents - 대화창의 항목들을 전달인자의 항목들로 갱신합니다.
on() String: eventName, Function: callback - 대화창의 이벤트를 리슨합니다.

예제

const dialog3 = CommonDialog("예제3", {
    contents: [
        {
            label: "팀명",
            text: "Scuderia Ferrari",
        },
    ],
});

dialog3.changeTitle("F1");

dialog3.setDataSource([
    {
        label: "팀명",
        text: " Red Bull Racing",
    },
    {
        label: "드라이버",
        text: "막스 베르스타펜, 세르히오 페레즈",
    },
]);
console.log(dialog3.getDataSource());

Event

항목 설명
onOpen 대화창 표시 이벤트
onClose 대화창 닫힘 이벤트
onEdit 대화창 편집 모드 진입 시 이벤트
onSave 저장 버튼 클릭 시 이벤트
onCancel 취소 버튼 클릭 시 이벤트

파라미터

dialog.on(eventName, callback);

    eventName: "onOpen" | "onClose" | "onEdit" | "onSave" | "onCancel"
    callback: Function // 이벤트 발생시 실행할 함수.

예제

dialog3.on("onOpen", () => {
    console.log("대화창 열림");
});

dialog3.on("onClose", () => {
    console.log("대화창 닫힘");
});
